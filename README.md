# Product Lab

## 生活认字

给家人使用的生活高频字词自学网页。

打开方式：

```text
D:\work\projects\product-lab\index.html
```

第一版内容：

- 今日学习：默认每天 5 个字词，可改成 3、5、8、10 个。
- 复习不会的：学错的字词会留下来。
- 生活场景：药品健康、手机付款、医院看病、出门坐车、买菜食品、家里生活。
- 语音带路：首页有“听我说”和一个大“开始”按钮。
- 固定音频：`audio/` 里放了字词和提示音频，手机不依赖浏览器自带朗读。

## 怎么给手机用

现在这个地址是本地文件，只能在这台电脑上打开。要给手机长期用，推荐把整个目录部署成静态网页：

1. Netlify、Vercel 或 GitHub Pages 都可以。
2. 部署后会得到一个固定网址。
3. 用她的手机浏览器打开这个网址，点“听我说”确认有声音。
4. 在浏览器菜单里选择“添加到桌面”。

第一阶段没有账号，学习记录会保存在她自己的手机里。
